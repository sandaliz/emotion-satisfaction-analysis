# Literature Notes - Music Emotion Research

## Aljanaki et al., 2014 - Technical Report
- **Purpose**: Describes Emotify game design and data collection methodology
- **Key insight**: Each line in the dataset = one participant's annotation (listening event)
- **Gender coding**: 1595 participants (651 F, 944 M) → confirms 0=Male, 1=Female
- **Important note**: No unique participant IDs in dataset

## Zentner et al., 2008 - GEMS Scale
- **Purpose**: Introduces Geneva Emotional Music Scales (GEMS)
- **9 emotion categories**: Amazement, Solemnity, Tenderness, Nostalgia, Calmness, Power, Joyful activation, Tension, Sadness
- **Superfactors**: Vitality, Sublimity, Unease
- **Relevance**: Our emotional intensity score is sum of these 9 binary emotions
